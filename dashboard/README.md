# Dashboard

## Purpose
This directory contains the web dashboard for visualizing recommendations.

## Framework Options
- **Streamlit**: Simple, Python-native approach
- **Dash**: More customizable, better for complex UIs

## Contents
- **app.py**: Main dashboard application
- **pages/**: Multi-page dashboard components (for Dash/Streamlit)
- **assets/**: CSS, images, and static files

## Running the Dashboard
```bash
# Streamlit
streamlit run dashboard/app.py

# Dash
python dashboard/app.py
```
