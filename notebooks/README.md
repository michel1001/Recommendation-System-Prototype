# Jupyter Notebooks

## Purpose
This directory contains exploratory analysis and development notebooks.

## Naming Convention
Use the format: `YYYY-MM-DD_author_topic.ipynb`

Examples:
- `2025-01-15_alice_data_exploration.ipynb`
- `2025-01-16_bob_model_testing.ipynb`

## Guidelines
- Keep notebooks for exploration and analysis only
- Move finalized code to appropriate modules in `src/`
- Document findings and decisions in markdown cells
- Clear output before committing large notebooks
